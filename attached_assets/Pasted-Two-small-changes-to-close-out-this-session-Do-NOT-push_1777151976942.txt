## Context

Manager cannot promote a user to admin or manager (only Admin can grant 
admin/manager roles). This matches the same restriction already in place on 
POST /api/invitations (server/routes.ts ~line 984).

Today, PATCH /api/users/:userId/role uses requireAdmin middleware, blocking Manager.

   standard or restricted roles".
