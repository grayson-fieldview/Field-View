# Context

Field View filters most queries by accountId for multi-tenant isolation
(every authenticated request scopes data to req.user.accountId). However, 
no indexes exist on accountId columns today. This means every query does 
a full table scan, which is fine at 2 users but will degrade fast as we 
add customers.

We're adding indexes to the highest-traffic tables tonight.

shared/models/auth.ts (which has an index on expire).

Tables that need indexes:

In shared/models/auth.ts:
- users table → index on accountId (named users_account_id_idx)

In shared/schema.ts:
