CONTEXT

- Hard delete: removes the media row, the S3 file, and ALL join rows pointing to it (checklist item refs, report section refs, task refs, etc.)
- 2-step confirm: if the photo is referenced anywhere, a second dialog lists the contexts ("This photo is in 1 report and 2 checklists. Are you sure?")
- Shared report warning: if the photo is in a report with an active share token, the dialog adds a stronger warning ("...this is a shared report — deleting will break the shared link.")

Web parity for the UI is NOT in scope this round. We're building the server endpoints only.
