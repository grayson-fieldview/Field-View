PART C — STORAGE METHODS (server/storage.ts)

Add three methods to IStorage:
- setProjectShareToken(projectId: number, accountId: string, 
    token: string | null): Promise<boolean>
  Account-scoped UPDATE. Returns true if updated, false if not 
  found/wrong account.
- getProjectByShareToken(token: string): Promise<Project | null>
  No auth — public lookup. Returns project row or null.
- getProjectPublicSummary(token: string): Promise<{
    project: { id, name, address, status, color, photoCount, 
      taskCount, completionPercent };
    account: { name, companyLogoUrl };
    coverPhoto: { url } | null;
    recentPhotos: Array<{ id, url, takenAt }>;  // last 6
  } | null>
  Joins project + account + media (cover + 6 most recent). All S3 
  URLs presigned. CAREFUL: filter out private fields — don't expose 
  createdById, internal notes, restricted-user assignments, etc. 
  Whitelist-only on the returned shape.

PART D — ROUTES (server/routes.ts)

Add four routes:

POST /api/projects/:id/share
- Auth: requireWriteAccess + verifyProjectAccess (existing middleware)
- Generates crypto.randomBytes(16).toString("base64url")
- Calls storage.setProjectShareToken(id, accountId, newToken)
- Returns 200 { shareToken: newToken }
- Mirror reports/share at server/routes.ts:1972-1985

DELETE /api/projects/:id/share
- Auth: same chain
- Calls storage.setProjectShareToken(id, accountId, null)
- Returns 200 { shareToken: null }
- Mirror reports/share DELETE at line 1988-2000

GET /api/public/projects/:token
- NO AUTH (token IS the credential)
- Calls storage.getProjectPublicSummary(token)
- Returns 404 if null
- 5-minute Cache-Control: public, max-age=300, s-maxage=300
- Mirror /api/public/reports/:token pattern

GET /api/public/projects/:token/cover.jpg
- NO AUTH
- Calls storage.getProjectByShareToken + look up cover media
- If no cover photo: 302 redirect to a default placeholder image URL 
  hosted in the public assets (use whatever existing default project 
  thumbnail the app uses; if none, redirect to a generic field-view 
  branded image)
- If cover photo exists: stream the S3 image bytes (NOT a redirect — 
  the image URL must be stable for OG scrapers). Set 
  Content-Type: image/jpeg, Cache-Control: public, max-age=86400.
- Implementation: fetch from S3 via existing helper, pipe to response
